%[if edit]%

This stack demonstrates the use use of "-addType" parameter to the %slice% template.

  • The outer stack creates an <ul> tag and allows only a specific type to be added.
  • Many inner stacks can be added by clicking the plus button.
  • The inner stacks can be added, removed, rearranged and moved between similar stacks.
  • The inner stacks can NOT be moved out of the outer stack, nor can any other stack be moved in.
  • The inner stack has a "hidden" property, so does not appear in the stacks library.
  • Each inner stack adds a <li> tag containing an image.
  • The inner stack has an "inline" property so will be exported without a stacks-out and stacks-in wrapper.
  • The %image% template is used to export a small and a large sized image.
  • Javascript and jQuery are used to add behavior to the exported images.

%[endif]% %[if !edit]%
